# 電腦軟體
## 辦公室與生產力工具
- **[7-Zip](https://www.7-zip.org/)** ![License: LGPL-2.1](https://img.shields.io/badge/License-LGPL--2.1-blue.svg) ![License: BSD-3-Clause](https://img.shields.io/badge/License-BSD--3--Clause-blue.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  開源、穩定、可靠的檔案壓縮工具，可惜處惟不像 WinRAR 有增強的修復功能。

- **[DocFetcher](https://docfetcher.sourceforge.io)** ![License: EPL](https://img.shields.io/badge/License-EPL-blue.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  開源的文件搜索器，相較於 Everything 以及內建的文件搜尋功能，它透過在指定搜尋範圍讀取文件（文件、簡報、PDF 等文件都可以讀取）建立索引，可透過搜尋檢索內文。  
  當記得電腦裡有某個文件但是不記得文件名稱時很方便使用。

- **[PDFGear](https://www.pdfgear.com/)** ![Free](https://img.shields.io/badge/Free-00ADD8?logo=free&logoColor=white) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![iOS](https://img.shields.io/badge/iOS-000000?logo=apple&logoColor=white) ![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)  
  免費的 PDF 閱讀與編輯軟體。  
  本人用作 Adobe Acrobat Pro 的替代品，提供了基本的 PDF 編輯功能、OCR、轉檔、壓縮等功能，以及近期整合 AI 對話等功能。功能完善，惟介面較為複雜，需要花一些時間適應，以及在一些特殊轉換需求較不完整。  本人日常使用其作為 PDF 編輯工具主力，而若有不足處則會使用 Ghostscript 輔助。  
  儘管如此，由於其付費選項、功能與時程尚未明訂，個人不建議應用於長期企業推廣使用，避免未來授權方式、政策、功能或時程改變影響使用。

- **[XnViewMP](https://www.xnview.com/en/xnviewmp/)** ![Free](https://img.shields.io/badge/Free-00ADD8?logo=free&logoColor=white) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  免費的照片檢視器，其支援多數影像格式。  
  特別是個人及執行計畫常會拍攝 RAW 格式的影像（DNG、CR2），以及有些人使用 iPhone 拍攝 HEIC 檔案格式，轉檔方面也支援常見的 JPEG、PNG、TIFF 等檔案格式。

## 生活品質與管理工具
### 筆記與知識管理
- **[Calibre](https://calibre-ebook.com/)** ![License: GPL-3.0](https://img.shields.io/github/license/kovidgoyal/calibre.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black) ![iOS](https://img.shields.io/badge/iOS-000000?logo=apple&logoColor=white) ![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)  
  開源的電子書管理器，支援主流的電子書格式。也有許多不錯的插件可以使用。  
  個人經常使用「Ebook Translator」插件翻譯電子書，翻譯效果個人可以接受，惟圖表非常容易跑版較為可惜。隨著 AI 翻譯技術進步，應該會越來越好。

- **[EndNote](https://endnote.com)** ![Paid](https://img.shields.io/badge/Paid-FF6B35?&logoColor=white) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white)  
  書目管理軟體，可將電子資料之元數據與全文系統化的儲存，並且可透過雲端帳號同步。  
  [EndNote 操作紀錄](3-EndNote日常操作)。

- **[Joplin](https://joplinapp.org/)** ![License: AGPL-3.0](https://img.shields.io/badge/License-AGPL--3.0-red.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![iOS](https://img.shields.io/badge/iOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black) ![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)   
  開源的筆記軟體，可在裡面使用 Markdown 進行格式化的筆記撰寫，並支援 KaTeX、Mermaid、MultiMarkdown 等語法。  
  功能方面對我來說已足夠日常使用。其中，個人偏愛其特點為可自行規劃筆記儲存位址（如雲端硬碟 OneDrive、Dropbox、Nextcloud，惟此做法會失去使用 Joplin Cloud 可以使用的分享筆記共同編輯的功能）並且具有端對端的安全性。惟個人認為較可惜處為不具備在本地端加密筆記的功能。

- **[Typora](https://typora.io/)** ![Paid](https://img.shields.io/badge/Paid-FF6B35?&logoColor=white) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  所見即所得、介面簡潔且易於使用的 Markdown 編輯器，支援多數 Markdown 語法以及 KaTeX、Mermaid 等語法。  
  個人用於次要的 Markdown 編輯器。

### 財務管理
- **[GnuCash](https://www.gnucash.org)** ![License: GPL-2.0](https://img.shields.io/badge/License-GPL--2.0-orange.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  開源的會計軟體，易於使用的複式記帳軟體，可以進行複式記帳、報表繪製，項目自訂性高。  
  惟沒有對行動平台的支援以及資料庫加密的功能較為可惜，不過由於其資料庫為一檔案，可自行運用其他方法進行加密與備份管理也不會過於複雜。

## 安全性工具
- **[Bitwarden](https://bitwarden.com/)** ![License: AGPL-3.0](https://img.shields.io/badge/License-AGPL--3.0-red.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black) ![iOS](https://img.shields.io/badge/iOS-000000?logo=apple&logoColor=white) ![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)  
  開源的密碼管理器，可用於保存密碼、身分、支付卡以及純文字信息，並且可透過雲端方式同步到其他設備。亦可自行託管組織伺服器。  
  在免費方案即可使用多數功能，惟 TOTP 屬付費功能，就算自行託管仍需付費解鎖。（個人使用付費方案，每年 10 美元真的很划算）  
  然而，在自動填充方面，個人認為 Bitwarden 支援度不如其他商業軟體（如 1Password、LastPass 等），有些網站會無法正確填充，或是需要手動選擇帳號密碼。

- **[KeePassXC](https://keepassxc.org/)** ![License: GPL-3.0](https://img.shields.io/badge/License-GPL--3.0-orange.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white) ![macOS](https://img.shields.io/badge/macOS-000000?logo=apple&logoColor=white) ![Linux](https://img.shields.io/badge/Linux-FCC624?logo=linux&logoColor=black)  
  開源的密碼管理器，其密碼資料庫為一檔案，可透過雲端方式同步使用，自訂性較高，大多也都有 TOTP 功能。  
  個人用於離線密碼管理，雖然主要密碼使用 Bitwarden，但是 TOTP 與密碼放在同一個地方總是比較不安心，因此使用 KeePassXC 來管理 TOTP。並且用另外一個 KeePassXC 資料庫來定期備份密碼資料庫。

- **[simplewall](https://github.com/henrypp/simplewall)** ![License: GPL-3.0](https://img.shields.io/github/license/henrypp/simplewall.svg) ![Windows](https://img.shields.io/badge/Windows-0078D4?logo=windows&logoColor=white)  
  開源的 Windows 防火牆軟體。與 Windows Firewall 同樣使用 Windows Filtering Platform (WFP) 進行通訊的過濾，然而介面簡單、軟體精簡也易於配置，也能同時與 Windows Firewall 運行。程式沒有啟動的情況也能保持防火牆規則。
  使用迄今認為主要有兩個缺點，一為無法透過如萬用字元方式設定規則，很多軟體在更新版本時會需要重新設定規則；二為該軟體仍使用自簽名數位簽章，這部分仍令人有所顧慮。

## 雲端/自架服務
### 自架服務
- **[Miniflux](https://miniflux.app/)** ![License: Apache-2.0](https://img.shields.io/github/license/miniflux/v2.svg) ![Self-hosted](https://img.shields.io/badge/Self--hosted-FF6B35?logo=docker&logoColor=white)
